# Blackjack

【概要】
JavaScriptを用いて制作したWebブラウザ版のブラックジャックです。
制作当時はHTMLを用いたUI制作が未経験でしたが、AIと対話しながら約6時間で共同制作し、
短期間で動作するプロダクトとして完成させるプロセスを経験しました。

【工夫したポイント】
1.複雑なルール判定の実装：エース（A）を状況に応じて1にも11にも変化させるロジックや、
ディーラーが合計17以上になるまでカードを引き続けるアルゴリズムを構築しました。
2.ゲーム性の追求：透視などの独自ルールを導入し、プログラム上での柔軟な条件設定を試みています。
3.学習意欲：未経験の領域でもツールを適切に活用し、形にする実行力を重視しました。

【改善点】
現在、PC本体のダークモード設定に依存して、ブラウザ側でトランプの白地が反転してしまう現象を確認しています。
この色の逆転を制御し、どのような閲覧環境でも正しく表示されるようCSSやメタタグの指定を最適化することが現在の目標です。

【実行方法】
1. リポジトリ内のGitHub Pagesリンクへアクセス
2. 「Blackjack=JSver」をクリックして実行
